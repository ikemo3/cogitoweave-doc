## 概要

セキュリティ・ネットワーク技術選択では、システムの安全性と可用性を確保するための技術スタックを選定する。ステップ 1・2 で構築された基盤インフラと監視環境を前提として、ネットワーク層でのセキュリティとパフォーマンス最適化を実現する具体的技術を決定する。
